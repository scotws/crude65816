Test Routines for the Crude 65816 Emulator

First version: 09. Okt 2015
This version: 22. Dec 2016

This folder contains various tests for the emulator. It's pretty unstructured and chaotic and probably confusing if you're not the author. At some point, this will be cleaned up properly.

- Tali Forth is a 8-bit Forth for the 65c02, see FEHLT. This is included as a gimick until a 16-bit Forth is avaiable. Change the configuration files to 

        putc f001
        getc f004
